Field Information
Required Fields
  • room_number - Unique identifier
  • name - Room name
Optional Fields
  • room_type - Type of room
  • capacity - Maximum occupancy
  • area_sqft - Area in square feet
  • description - Room description
  • is_active - Status (TRUE/FALSE)
Room Types
office meeting storage server conference hall chamber lobby utility washroom pantry oath other
Import Instructions
  1. Download Template: Use the provided template to format your data correctly.
  2. Required Fields: room_number and name are mandatory fields.
  3. Room Number: Must be unique. Existing numbers will be updated if "Update existing" is checked.
  4. Room Type: Use valid room types from the list. Defaults to 'office' if not specified.
  5. Capacity: Enter positive integers only for room capacity.
  6. Area: Enter positive numbers (can include decimals) for square footage.
  7. Status Field: Use TRUE/FALSE, YES/NO, or 1/0 for the is_active field.
  8. Review Results: Check the import summary and address any errors reported.

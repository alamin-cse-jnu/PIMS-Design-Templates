Field Information
Required Fields
  • office_code - Unique identifier
  • name - Office name
Optional Fields
  • office_type - Type of office
  • head_of_office - Officer in charge
  • contact_number - Phone number
  • email - Email address
  • description - Office description
  • is_active - Status (TRUE/FALSE)
Office Types
wing branch section secretary speaker deputy speaker chief Whip whip chairman mp unit store project other
Import Instructions
  1. Download Template: Use the provided template to format your data correctly.
  2. Required Fields: office_code and name are mandatory fields.
  3. Office Code: Must be unique. Existing codes will be updated if "Update existing" is checked.
  4. Office Type: Use valid office types from the list. Defaults to 'wing' if not specified.
  5. Email Format: Must be valid email addresses (e.g., user@domain.com).
  6. Phone Numbers: Can be in any format. Enable normalization for consistent formatting.
  7. Status Field: Use TRUE/FALSE, YES/NO, or 1/0 for the is_active field.
  8. Review Results: Check the import summary and address any errors reported.
